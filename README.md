\## הסקריפטים שנכתבו על ידי עידו

1\. \*\*בדיקת מזג אוויר (weather.sh)\*\* - שולף את מזג האוויר לעיר מבוקשת.

2\. \*\*יצירת סיסמה (generate\_password.sh)\*\* - מפיק סיסמה רנדומלית מאובטחת בעלת 10 תווים.

3\. \*\*גיבוי תיקיה (backup\_folder.sh)\*\* - יוצר קובץ גיבוי דחוס (tar.gz) תכולת תיקיה.

4\. \*\*ספירת קבצים (count\_extensions.sh)\*\* - סופר כמה קבצים יש בתיקיה מסוימת לפי סיומת.

5\. \*\*זמן פעילות (system\_uptime.sh)\*\* - מציג את זמן הפעילות הנוכחי של המערכת.



\## הוראות הרצה למשתמש

כדי להריץ את הסקריפטים, יש לבצע את השלבים הבאים לפי הסדר:



\### שלב א': שכפול הפרויקט (Clone)

יש להוריד את הפרויקט למחשב המקומי באמצעות הפקודה:

`git clone https://github.com/Afeka-DevTools/26b-10142-bash-ido\_veronika\_oriyan.git`

ולאחר מכן להיכנס לתיקייה:

`cd 26b-10142-bash-ido\_veronika\_oriyan`



\### שלב ב': מתן הרשאת הרצה (חובה)

לפני הרצת סקריפט בפעם הראשונה, יש לתת לו הרשאת הרצה:

`chmod +x scripts/שם\_הסקריפט.sh`



\### שלב ג': הרצה ומתן קלט

\* \*\*weather.sh\*\*: הרץ `./scripts/weather.sh` והכנס את שם העיר כשתתבקש (למשל: Tel-Aviv).

\* \*\*generate\_password.sh\*\*: הרץ `./scripts/generate\_password.sh` (הסקריפט ירוץ וידפיס את הסיסמה ללא צורך בקלט).

\* \*\*backup\_folder.sh\*\*: הרץ `./scripts/backup\_folder.sh` והכנס את הנתיב המלא לתיקיה שתרצה לגבות.

\* \*\*count\_extensions.sh\*\*: הרץ `./scripts/count\_extensions.sh`, הכנס נתיב לתיקיה, ולאחר מכן הכנס סיומת קובץ (למשל: txt).

\* \*\*system\_uptime.sh\*\*: הרץ `./scripts/system\_uptime.sh` (אין צורך בקלט).
